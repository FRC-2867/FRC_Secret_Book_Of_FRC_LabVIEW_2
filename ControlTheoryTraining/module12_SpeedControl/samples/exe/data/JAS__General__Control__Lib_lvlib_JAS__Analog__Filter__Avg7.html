JAS_General_Control_Lib.lvlib:JAS_Analog_Filter_Avg7.vi

This VI averages the last seven samples of an analog value. The sample rate is determined by how often this VI is called. No consideration is given to the time between samples. Filtering can be enabled or disabled. When disabled the input value is passed directly to the output

 Value_Input

The input value (Required)

Enable

Enable filtering. If disabled the input value is passed directly to the output. Default: Enabled. (Optional)

Value_Output

Output value. When filtering is enabled this is the filtered value. When disabled this is equal to the input.
